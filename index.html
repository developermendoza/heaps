Heaps

A heap is specialized tree-based data structure that satisfies the heap property. In a heap, for every node 'i' other than the root:

  • If 'i' is a max-heap, the value of 'i' is greater than or equal to the values of its children
  • If 'i' is a min-heap, the value of 'i' is less than or equal to the values of its childres.

Heaps are commonly used to implement priority queues, where elements with heigher priority (or lower priority, depending on the implementation) are served before elements with lower priority

There are two main types of heaps:

  1. Max Heap: The value of each node is greater than or equal to the values of its children.
  2. Min Heap: The value of each node is less than or equal to the values of its children.
